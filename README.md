# Flask Todo App — Dockerized with PostgreSQL

A simple Todo List web application built with Flask and PostgreSQL, fully containerized using Docker and Docker Compose, deployed on AWS EC2.

## Project Overview

| Technology | Use |
|---|---|
| Python Flask | Web application |
| PostgreSQL | Database |
| Docker | Containerization |
| Docker Compose | Multi-container management |
| AWS EC2 | Cloud deployment |

## Features
- Add new todos
- Delete todos
- Data stored in PostgreSQL database
- Fully containerized with Docker
